Esque- A Fictional Waxing Brand

This project was one of my Final First Year Projects. The project entails a fictional brand that was inspired by other waxing brands. Our assignmentâ€™s aim was to create a packaging for a grooming product. Esque is a fictional brand. It is designed as a vegan friendly waxing brand that comes in three different sizes, tub which is for salon based waxing, tube wax which is for portability and travel and finally wax strips that allow for a smooth waxing experience. The packaging was designed on illustrator. To create a rather wholesome look, I used vector graphics to add specific visual details to the packaging. The illustrators were done using the pen and pencil tools.
